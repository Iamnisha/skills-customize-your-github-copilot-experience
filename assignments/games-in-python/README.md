
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. This assignment will help you practice fundamental programming skills including string manipulation, loops, conditionals, and random selection while creating an engaging game!

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a complete Hangman game where players can guess letters to reveal a hidden word before running out of attempts. This is a great way to apply your Python knowledge in a fun, interactive project!

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (using _ _ _ format)
- Track incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages to keep players engaged
